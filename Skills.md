# 自动驾驶感知算法工程师研究 Skills (Ultimate Edition)

## 技能 1：极速高保真文献解构 (High-Fidelity Analysis)
- **获取策略**：优先解析 **ar5iv (HTML)** 以获取精准的数学公式（MathML）和高清图表直链，规避慢速 PDF 下载。
- **物理还原**：具备推导相机投影几何、单应性变换（Homography）、3D 位置编码（PE）及自车运动补偿（Ego-motion）的数学建模能力。

## 技能 2：源码驱动的深度审计 (Code-Driven Audit)
- **云端审计**：直接通过 GitHub Raw 或克隆极小深度仓库，深入 `forward` 函数、`Loss` 计算逻辑及 `Data Augmentation` 脚本。
- **伪代码建模**：将抽象公式转化为带有 **Tensor Shape (张量形状)** 标注的 PyTorch 风格伪代码，实现“代码即真相”。

## 技能 3：全栈感知演进流水线 (Full-Stack Path)
- **知识闭环**：管理从 Level 01 (单目几何) 到 Level 05 (生成式世界模型) 的演进树。
- **部署预研**：评估算子在 Orin-X/BPU 等算力平台上的访存瓶颈（Memory-bound）及 TensorRT 量化友好度。

## 技能 4：自动化知识固化与可视化
- **结构化存档**：维护 10 步极致硬核速读模版。
- **拓扑可视化**：具备生成 3D 演进关系图谱（Plotly/NetworkX）的能力，宏观把控技术流派。
